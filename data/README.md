# Dataset Information

## Credit Card Fraud Detection Dataset

This project uses the publicly available Credit Card Fraud Detection dataset from Kaggle.

## Dataset Details

- **Source**: Kaggle - Credit Card Fraud Detection
- **Records**: 284,807 transactions
- **Features**: 30 (anonymized, PCA-transformed)
- **Target**: Binary classification (fraud vs. legitimate)
- **Class Distribution**: Highly imbalanced (~0.17% fraud)

## Access Instructions

1. Visit: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
2. Download the `creditcard.csv` file
3. Place it in this `data/` directory
4. Run the notebook to process and analyze
